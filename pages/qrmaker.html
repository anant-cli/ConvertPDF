QR code generator

Create QR codes for links, text, contact details, and simple sharing tasks. The code is generated in your browser and can be downloaded for print or web use.

Best for: event links, menus, Wi-Fi cards, payment pages, and posters.
Input: text or URLs.
Output: PNG or SVG QR codes.

Before you download

Test the QR code with a phone before printing. High contrast and enough quiet space around the code improve scan reliability.

Privacy

The selected file is handled in your browser. It is not uploaded to ConvertPDF servers, and the downloaded result is created on your device.
