Password protect PDF

Add a password to a PDF before sharing it. You can also control common permissions such as printing, copying, and editing.

Best for: contracts, financial files, HR records, client drafts, and private forms.
Input: PDF files.
Output: an encrypted PDF.

Before you download

Use a strong password and store it safely. ConvertPDF cannot recover it later because the file is processed locally.

Privacy

The selected file is handled in your browser. It is not uploaded to ConvertPDF servers, and the downloaded result is created on your device.
