About Me

I built ConvertPDF to solve a real problem I encountered daily: needing to convert documents — Markdown notes, Word files, images — without uploading them to unknown servers. Most “free” online converters require you to hand your files to a third-party server. That never sat right with me.

So I built a solution: a fully client-side PDF toolkit that runs entirely in your browser. No uploads. No tracking. No limits. Open source.

The Project

ConvertPDF is hosted at convert-pdf-tool.netlify.app and includes tools for Markdown-to-PDF, DOCX-to-PDF, image conversion, PDF merging, password protection, QR code generation, and more — all processed locally in your browser using modern JavaScript libraries.

The full source code is available on GitHub.
