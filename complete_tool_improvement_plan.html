✍️ Sign PDF
2 Bugs3 features
▼
Current bugs
BugCanvas has background: white in CSS. canvas.toDataURL('image/png') includes that white background. On white PDFs this is invisible; on coloured PDFs it looks broken. Fix: call ctx.clearRect() before drawing, never fill white, so the PNG has a transparent background.
BugText positioning uses text.length * fontSize * 0.6 — this is wrong. pdf-lib draws text left-anchored from x. For bottom-right the x should be width - estimatedTextWidth - margin. The rough formula overflows for long names and misaligns for short ones.
BugDrawn signature check: imageData.data.some(pixel => pixel !== 0) — checks all 4 channels (RGBA). An empty transparent canvas has all zeros so this works, but if user clears with clearRect after drawing, alpha channel gets reset properly. Logic is fine but fragile.
Real solution
FixRemove background: white from canvas style. The canvas 2D context is transparent by default. PNG export will be transparent. In pdf-lib, transparent PNG embeds correctly — only the ink shows.
FixUse pdf-lib's StandardFonts.Helvetica to embed font, measure text width accurately with font.widthOfTextAtSize(text, size), then compute x position correctly for all 4 corners.
FeatureAdd signature color picker (black/blue/red ink) — applies to both text and drawn signatures.
FeatureAdd 3 typed signature font styles using pdf-lib StandardFonts: Helvetica (clean), TimesRoman (formal), Courier (typewriter).
FeatureAdd a live preview: render PDF page 1 via pdf.js, overlay the signature position as a visual preview before downloading.
pdf-lib StandardFonts + font.widthOfTextAtSize() — no extra library needed

💧 Watermark PDF
Bug2 features
▼
Current bug
BugCanvas preview rotation: code calls ctx.rotate(Math.PI / 4) then ctx.fillText(x, y) but rotation is around origin (0,0), not around the text center. Result: text renders off-page or in wrong position. Fix needs ctx.save() → ctx.translate(cx, cy) → ctx.rotate(angle) → ctx.fillText(0, 0) → ctx.restore().
Real solution
FixCorrect canvas rotation: ctx.save(); ctx.translate(x, y); ctx.rotate(Math.PI/4); ctx.fillText(text, 0, 0); ctx.restore(); — this rotates around the text's own center point.
FeatureAdd image watermark: let user upload a PNG logo. Embed it into pdf-lib with pdfDoc.embedPng(), then page.drawImage() with controllable opacity and position. Real use-case for businesses.
FeatureAdd "Tile" mode: repeat watermark across the whole page in a grid pattern. Compute grid positions based on page size and watermark text width, loop page.drawText() at each grid point.
